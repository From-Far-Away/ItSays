# IT SAYS - SERVER RESTFUL FOR SAY IT! ANDROID APP

## Installation procedure

In any folder :
```
git clone git@github.com:From-Far-Away/ItSays.git
cd ItSays
npm install
echo "DB_HOST=<your_host>\r\nDB_USER=<your_user_credentials>\r\nDB_PASSWORD=<password>" >> .env
```

And to run it :
```
node server.js
```
